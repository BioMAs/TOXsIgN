Search

Search engineuse

The search engine supports complex fielded queries to retrieve signatures and associated data based on various information such as experimental parameters, observed toxicological effects and/or up/down-regulated gene lists. This module is especially powerfull for identifying comparable experiments (i.e. performed on similar models and/or with close environmental factors) and/or similar outcomes (i.e. close physiological, molecular and/or omics effects).

Browse ontologiesuse

TOXsIgN also includes a tool to browse the ontologies used in the resource to describe data. This feature is especially relevant for filling the excel template required when submitting a novel signature. This tool use the NCBI OntologyTree widget from BioPortal. It is configured to display all the ontologies used in TOXsIgN.

Convert

Conversion tooluse

The conversion tool allows users to convert a list of EntrezGene Ids or HomoloGene Ids to HomoloGene Ids or EntrezGene Ids.

Compare

Signature Comparison (Venn diagram)use

The comparison module allows users to compare several toxicogenomic signatures It is based on the tool developed by P. Bardou et al that can display intersection between up to 6 signatures in a Venn diagram.

Signature enrichment analysisuse

The signature enrichment analysis module allows users to compare a selected toxicogenomic signatures to the overall set of signatures deposited by the community using a distance analysis approach (based on the Euclidean distance and the correlation coefficient).

Analyze

Functional enrichment analysisuse

The functional enrichment analysis allows users to identify biological processes, molecular functions, cellular components and phenotypes significantly associated to a selected signature.

Signature enrichment analysisuse

The signature enrichment analysis module allows users to compare a selected toxicogenomic signatures to the overall set of signatures deposited by the community using a distance analysis approach (based on the Euclidean distance and the correlation coefficient).

Extract

Extract toxicogenomic signature (under construction)

This module allows users to automatically extract the toxicogenomic signature from a transcriptomic dataset (Affymetrix GeneChip). This tool is under construction and will be available soon.

Predict

ChemPSy - Prediction tool (under construction)

This module will allow users to predict the adverse effects of a given chemical compound based on the approaches developed in the ChemPSy project. The aim of ChemPSy (Chemical Prioritization System) is to provide an innovative tool based on several bioinformatics and biostatistics methodologies to analyze and integrate massive toxicogenomic datasets. Specific objectives include: (1) classification of chemicals based on transcriptional signatures, i.e. the set of genes whose expression is known to be positively or negatively altered after an exposure to these compounds; (2) the association of classes with human pathologies or deleterious phenotypes, i.e. classes containing toxicants with well-known effects; (3) the prediction of novel reprotoxicants and/or endocrine disruptors based on transcriptional signature similarities with known chemicals affecting testis development and function. This tool is under construction and will be available soon.
