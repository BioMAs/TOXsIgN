Upload signature or additionnal file

In order to upload a file to your signature, navigate to your signature view (http://toxsign.genouest.org/app/#/browse?dataset=TSSXXXX) and click the upload button corresponding to your file in the Upload section in the bottom of the page.

In TOXsIgN your are able to upload 4 types of files :

  • Additional file : Any type of file related to the signature
  • Genes up file : [Only for genomics signatures] List of over expressed genes.
  • Genes down file : [Only for genomics signatures] List of under expressed genes.
  • Genes interrogated file : [Only for genomics signatures] List of all interrogated genes.
If you have already associated a file to your signature via the TOXsIgN spreadsheet, the uploaded file must have the same name.
Note : Genomics files must be in the following format : text file with one gene ID / line. Don't use Excel to create your text file.
